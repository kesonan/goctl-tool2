# goctl-tool

## 概述
`goctl-tool` 是为 `go-zero` 框架设计的一组工具链插件，旨在增强 Go 开发能力。

## 功能

### goctl API 生成

不用担心不会写api文件，goctl 在网页端轻松构建提供了 form 表单形式辅助你快速构建 api 文件。

- 支持请求体，响应体从 json 文件导入
- 请求体自主填写
- 支持 options 配置
